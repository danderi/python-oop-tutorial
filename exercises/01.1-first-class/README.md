# Creating Your First Class

## 📝 Instructions

1. Create a class called `Person` in the `app.py` file
2. Add an `__init__` method that takes `name` as a parameter
3. Store the name in an instance variable
4. Add a method called `greet` that prints "Hello, my name is {name}"

## 💡 Hints

- Use the `class` keyword to define a class
- The `__init__` method is the constructor
- Use `self` to refer to instance variables
