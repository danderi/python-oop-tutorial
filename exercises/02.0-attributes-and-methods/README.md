# Understanding Class Parts: Attributes and Methods 🧩

Hey there! Now that you've created your first class, let's talk about the two main parts that make up a class: attributes and methods. Don't worry - they're simpler than they sound!

## Think of it Like a Toy Robot 🤖

Imagine you have a toy robot:
- **Attributes** are like the robot's features (its color, height, battery level)
- **Methods** are like the things the robot can do (dance, speak, move forward)

## Two Types of Each! 📦

1. **Instance Attributes and Methods**
   - These belong to each individual thing we create
   - Like how each robot has its own battery level
   - We create these using `self` inside `__init__`

2. **Class Attributes and Methods**
   - These are shared by all instances of the class
   - Like how all robots from the same factory might share the same model number
   - We create these outside of `__init__`

## Real-World Example 🌟

Think about students in a school:
- **Instance Attributes**: Each student's name, age, and grades (different for each student)
- **Class Attributes**: School name, school mascot (same for all students)
- **Instance Methods**: Taking a test, doing homework (actions each student does)
- **Class Methods**: Announcing a school holiday (affects all students)
