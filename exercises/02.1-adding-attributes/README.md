# Working with Class and Instance Attributes

## 📝 Instructions

1. Create a class called `Student` in `app.py`
2. Add a class attribute `school_name` set to "Python Academy"
3. Add instance attributes in `__init__`:
   - name
   - age
   - grade
4. Create a method called `student_info` that returns all information about the student

## 💡 Hints

- Class attributes are defined outside `__init__`
- Instance attributes are defined inside `__init__`
- Use f-strings for the `student_info` method
