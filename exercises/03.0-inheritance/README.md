# Understanding Inheritance in Python

Inheritance is a fundamental concept in Object-Oriented Programming that allows a class to inherit attributes and methods from another class. This creates a parent-child relationship between classes.

## Key Concepts

1. **Base Class (Parent Class)**: The class being inherited from
2. **Derived Class (Child Class)**: The class that inherits from another class
3. **Method Override**: Child classes can provide specific implementation of methods
4. **super()**: Used to call methods from the parent class

## Why Use Inheritance?

- Promotes code reuse
- Creates a hierarchical relationship between classes
- Allows for method overriding and extension
- Helps in implementing polymorphism
